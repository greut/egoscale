exo config delete

Delete an account from config file

exo config delete <account name> [flags]

Options

--force, -f - Attempt to remove an account without prompting for confirmation
--help, -h - help for delete

Options inherited from parent commands

--account, -A - Account to use in config file [env EXOSCALE_ACCOUNT]
--config, -C - Specify an alternate config file [env EXOSCALE_CONFIG]
