exo dns add MX

Add MX record type to a domain

Synopsis

Add a mail exchange record that points to a mail server or relay.
These types of records are used to describe which servers handle incoming email.

exo dns add MX <domain name> [flags]

Options

--help, -h - help for MX
--mail-server-host, -m - Example: mail-server.example.com
--name, -n - Leave this blank to create a record for <domain name>
--priority, -p - Common values are for example 1, 5 or 10
--ttl, -t - The time in second to leave (refresh rate) of the record.

Options inherited from parent commands

--account, -A - Account to use in config file [env EXOSCALE_ACCOUNT]
--config, -C - Specify an alternate config file [env EXOSCALE_CONFIG]
