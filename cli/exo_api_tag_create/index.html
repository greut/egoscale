exo api tag create

Creates resource tag(s)

Synopsis

Creates resource tag(s) http://cloudstack.apache.org/api/apidocs-4.4/user/createTags.html

exo api tag create [flags]

Options

--resourceids - (required) list of resources to create the tags for
--resourcetype - (required) type of the resource
--tags - (required) Map of tags (key/value pairs)
--customer - identifies client specific tag. When the value is not null, the tag can't be used by cloudStack code internally
--help, -h - help for create

Options inherited from parent commands

--account, -A - Account to use in config file [env EXOSCALE_ACCOUNT]
--config, -C - Specify an alternate config file [env EXOSCALE_CONFIG]
--debug, -d - debug mode on
--dry-run, -D - produce a cURL ready URL
