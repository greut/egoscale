exo privnet update

Update a private network

exo privnet update <name | id> [flags]

Options

--description, -d - Private network description
--endip, -e - the ending IP address in the network IP range. Required for managed networks.
--help, -h - help for update
--name, -n - Private network name
--netmask, -m - the netmask of the network. Required for managed networks
--startip, -s - the beginning IP address in the network IP range. Required for managed networks.

Options inherited from parent commands

--account, -A - Account to use in config file [env EXOSCALE_ACCOUNT]
--config, -C - Specify an alternate config file [env EXOSCALE_CONFIG]
