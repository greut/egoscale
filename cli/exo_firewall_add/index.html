exo firewall add

Add rule to a security group

exo firewall add <security group name | id>  [ssh | telnet | rdp | ...] (default preset rules) [flags]

Options

--cidr, -c - Rule Cidr [CIDR 0.0.0.0/0,::/0,…]
--description, -d - Rule description
--egress, -e - By default rule is INGRESS (set –egress to have EGRESS rule)
--help, -h - help for add
--icmp-code - Set icmp type code
--icmp-type - Set icmp type
--ipv6, -6 - Set ipv6 on default rules or on –my-ip
--my-ip - Set CIDR for my ip
--port, -P - Rule port range [80-80,443,22-22]
--protocol, -p - Rule Protocol available [tcp, udp, icmp, icmpv6, ah, esp, gre]
--security-group, -s - Rule security group [name or id ex: sg1,sg2…]

Options inherited from parent commands

--account, -A - Account to use in config file [env EXOSCALE_ACCOUNT]
--config, -C - Specify an alternate config file [env EXOSCALE_CONFIG]
