Egoscale

the Go library for Exoscale

egoscale

the Go library powering the docker machine driver, the Let's Encrypt client in Go (lego), the Terraform provider for Exoscale and many others.

  • Kubernetes Incubator: External DNS
  • Docker machine
  • etc.
