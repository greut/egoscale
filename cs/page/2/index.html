cs: Command-Line Interface for CloudStack

Like the Pythonic cs but in Go.

Installation

Grab it from the release section.

or build it yourself, it requires somes dependencies though to be pulled using dep.

Configuration

Create a config file cloudstack.ini or $HOME/.cloudstack.ini.

; Default region
[cloudstack]

; Exoscale credential
endpoint = https://api.exoscale.ch/compute
key = EXO...
secret = ...

theme = fruity


; Another region
[cloudstack:production]

endpoint = https://api.exoscale.ch/compute
key = EXO...
secret = ...

theme = vim


; global config for themes
[exoscale]
; Pygments theme, see: https://xyproto.github.io/splash/docs/
; dark
theme = monokai
; light
theme = tango
; no colors (only boldness is allowed)
theme = nocolors

Themes

Thanks to Alec Thomas’ chroma, it supports many themes for your output:

  • https://xyproto.github.io/splash/docs/
  • https://help.farbox.com/pygments.html

Usage

Some of the flags around a command.

$ cs (-h | --help)              global help
$ cs <command> (-h | --help)    help of a specific command
$ cs <command> (-d | --debug)   show the command and its expected output
$ cs <command> (-D | --dry-run) show the signed command
$ cs (-r | --region) <region>   specify a different region, default `cloudstack`

Commands

  • cs deleteReverseDnsFromPublicIpAddress

    delete the PTR DNS record from the public IP address <http://cloudstack.apache.org/api/apidocs-4.4/user/deleteReverseDnsFromPublicIpAddress.html>

  • cs deleteReverseDnsFromVirtualMachine

    Delete the PTR DNS record(s) from the virtual machine <http://cloudstack.apache.org/api/apidocs-4.4/user/deleteReverseDnsFromVirtualMachine.html>

  • cs deleteSSHKeyPair

    Deletes a keypair by name <http://cloudstack.apache.org/api/apidocs-4.4/user/deleteSSHKeyPair.html>

  • cs deleteSecurityGroup

    Deletes security group <http://cloudstack.apache.org/api/apidocs-4.4/user/deleteSecurityGroup.html>

  • cs deleteSnapshot

    Deletes a snapshot of a disk volume. <http://cloudstack.apache.org/api/apidocs-4.4/user/deleteSnapshot.html>

  • cs deleteTags

    Deleting resource tag(s) <http://cloudstack.apache.org/api/apidocs-4.4/user/deleteTags.html>

  • cs deleteUser

    Deletes a user for an account <http://cloudstack.apache.org/api/apidocs-4.4/root_admin/deleteUser.html>

  • cs deployVirtualMachine

    Creates and automatically starts a virtual machine based on a service offering, disk offering, and template. <http://cloudstack.apache.org/api/apidocs-4.4/user/deployVirtualMachine.html>

  • cs destroyVirtualMachine

    Destroys a virtual machine. <http://cloudstack.apache.org/api/apidocs-4.4/user/destroyVirtualMachine.html>

  • cs disableAccount

    Disables an account <http://cloudstack.apache.org/api/apidocs-4.4/root_admin/disableAccount.html>

  • cs disassociateIpAddress

    Disassociates an ip address from the account. <http://cloudstack.apache.org/api/apidocs-4.4/user/disassociateIpAddress.html>

  • cs enableAccount

    Enables an account <http://cloudstack.apache.org/api/apidocs-4.4/root_admin/enableAccount.html>

  • cs expungeVirtualMachine

    Expunge a virtual machine. Once expunged, it cannot be recoverd. <http://cloudstack.apache.org/api/apidocs-4.4/user/expungeVirtualMachine.html>

  • cs getVMPassword

    Returns an encrypted password for the VM <http://cloudstack.apache.org/api/apidocs-4.4/user/getVMPassword.html>

  • cs getVirtualMachineUserData

    Returns user data associated with the VM <http://cloudstack.apache.org/api/apidocs-4.4/user/getVirtualMachineUserData.html>

  • cs listAccounts

    Lists accounts and provides detailed account information for listed accounts <http://cloudstack.apache.org/api/apidocs-4.4/user/listAccounts.html>

  • cs listAffinityGroups

    Lists affinity groups <http://cloudstack.apache.org/api/apidocs-4.4/user/listAffinityGroups.html>

  • cs listApis

    lists all available apis on the server <http://cloudstack.apache.org/api/apidocs-4.4/user/listApis.html>

  • cs listAsyncJobs

    Lists all pending asynchronous jobs for the account. <http://cloudstack.apache.org/api/apidocs-4.4/user/listAsyncJobs.html>

  • cs listEvents

    A command to list events. <http://cloudstack.apache.org/api/apidocs-4.4/user/listEvents.html>
