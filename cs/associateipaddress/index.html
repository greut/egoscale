cs associateIpAddress

Acquires and associates a public IP to an account. http://cloudstack.apache.org/api/apidocs-4.4/user/associateIpAddress.html

Flags

--account value – the account to associate with this IP address
--domainid value – the ID of the domain to associate with this IP address
--fordisplay value – an optional field, whether to the display the ip to the end user or not (default: unset)
--isportable value – should be set to true if public IP is required to be transferable across zones, if not specified defaults to false (default: unset)
--networkid value – The network this ip address should be associated to.
--regionid value – region ID from where portable ip is to be associated. (default: 0)
--zoneid value – the ID of the availability zone you want to acquire an public IP address from

Global Flags

--debug, -d – debug mode on
--region value, -r value – cloudstack.ini file section name
--theme value, -t value – syntax highlighting theme, see: https://xyproto.github.io/splash/docs/
--help, -h – show help
--version, -v – print the version
