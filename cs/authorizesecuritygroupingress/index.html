cs authorizeSecurityGroupIngress

Authorizes a particular ingress/egress rule for this security group http://cloudstack.apache.org/api/apidocs-4.4/user/authorizeSecurityGroupIngress.html

Flags

--account value – an optional account for the security group. Must be used with domainid.
--cidrlist value – the cidr list associated
--description value – the description of the ingress/egress rule
--domainid value – an optional domainid for the security group. If the account parameter is used, domainid must also be used.
--endport value – end port for this ingress rule (default: 0)
--icmpcode value – error code for this icmp message (default: 0)
--icmptype value – type of the icmp message being sent (default: 0)
--protocol value – TCP is default. UDP, ICMP, ICMPv6, AH, ESP, GRE are the other supported protocols
--securitygroupid value – The ID of the security group. Mutually exclusive with securitygroupname parameter
--securitygroupname value – The name of the security group. Mutually exclusive with securitygroupid parameter
--startport value – start port for this ingress rule (default: 0)
--usersecuritygrouplist value – user to security group mapping
